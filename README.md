### Installation

Step by step follow me :

1. Clone the repo
   ```sh
   git clone https://github.com/A-SED-HASAN/sudoku
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. start local
   ```sh
   npm start
   ```
